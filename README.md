# 🛍️ FutureShop Landing Page

Landing page moderna desenvolvida para a FutureShop, focada em apresentação de produtos, experiência visual profissional e alta conversão.

- CSS3

## ⚙️ Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/Adriano-Dev22/landing-page-futureshop.git
```

### 2. Entrar na pasta do projeto

```bash
cd loja
```

### 3. Criar ambiente virtual

```bash
python -m venv venv
```

### 4. Ativar ambiente virtual

#### Windows (PowerShell)

```powershell
.\venv\Scripts\Activate.ps1
```

#### Windows (CMD)

```cmd
venv\Scripts\activate.bat
```

### 5. Instalar dependências

```bash
pip install flask
```

### 6. Executar aplicação

```bash
python app.py
```
